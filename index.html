url-tetris

This is a version of the popular Tetris game running in your address bar!

CONTROLS

W or ⬆️ Move upwards
A or ⬅️ Move leftwards
S or ⬇️ Move downwards
D or ➡️ Rotate
SPACE Start the game

HOW DOES IT WORK

With the History.replaceState() method the current url can be changed without reloading the page. For the graphics Braille Patterns are used.

Inspired by URL Hunter and Flappy Braille.
